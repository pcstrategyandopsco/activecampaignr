# Create a Contact

Create a Contact

## Usage

``` r
ac_create_contact(
  email,
  first_name = NULL,
  last_name = NULL,
  phone = NULL,
  ...
)
```

## Arguments

- email:

  Email address (required)

- first_name:

  First name

- last_name:

  Last name

- phone:

  Phone number

- ...:

  Additional fields

## Value

A single-row tibble of the created contact
